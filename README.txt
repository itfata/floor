Готовий сайт: index.html

Це окремий лендинг під модульну пластикову підлогу.
Без логотипа і без форми заявок.

Як замінити номер телефону:
1. Відкрийте index.html.
2. Унизу файла знайдіть:
   const PHONE = '+38 (XXX) XXX-XX-XX';
   const PHONE_TEL = '+380000000000';
3. У PHONE вставте номер для показу на сайті.
4. У PHONE_TEL вставте номер цифрами для кліку з телефона.

Приклад:
const PHONE = '+38 (067) 123-45-67';
const PHONE_TEL = '+380671234567';
